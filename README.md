This repository contains the data, scripts, and analysis workflows associated with our study on the molecular mechanism of urea-induced destabilization of RNA structures, with a focus on the SARS-CoV-2 RNA frameshift stimulation element (FSE).
Associated Publication
This repository accompanies the manuscript: Dissecting Urea-RNA Interactions in SARS-CoV-2 Virus: Nucleobases as the Primary Denaturation Hotspots (Submitted, 2026)
